Это 1583 год <a href="/wiki/%D0%9D%D0%B0%D1%88%D0%B0_%D1%8D%D1%80%D0%B0" title="Наша эра">нашей эры</a>, 3 год 9-го десятилетия <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI века</a> <a href="/wiki/2-%D0%B5_%D1%82%D1%8B%D1%81%D1%8F%D1%87%D0%B5%D0%BB%D0%B5%D1%82%D0%B8%D0%B5" title="2-е тысячелетие">2-го тысячелетия</a>, 4 год <a href="/wiki/1580-%D0%B5_%D0%B3%D0%BE%D0%B4%D1%8B" title="1580-е годы">1580-х годов</a>. <h2><span id=".D0.A1.D0.BE.D0.B1.D1.8B.D1.82.D0.B8.D1.8F"></span><span class="mw-headline" id="События">События</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=1583_%D0%B3%D0%BE%D0%B4&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Редактировать раздел «События»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=1583_%D0%B3%D0%BE%D0%B4&amp;action=edit&amp;section=1" title="Редактировать раздел «События»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>Январь&#160;— <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%81%D1%83%D0%B0_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%90%D0%BB%D0%B0%D0%BD%D1%81%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8_%D0%90%D0%BD%D0%B6%D1%83%D0%B9%D1%81%D0%BA%D0%B8%D0%B9)" title="Франсуа (герцог Алансонский и Анжуйский)">Герцог Франсуа Анжуйский</a>, опираясь на наёмные французские войска, поднял мятеж с целью присоединения <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Нидерланды">Нидерландов</a> к <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Королевство Франция">Франции</a>. Мятеж подавлен. Испанским наместником назначен Александр <a href="/wiki/%D0%94%D0%BE%D0%BC_%D0%A4%D0%B0%D1%80%D0%BD%D0%B5%D0%B7%D0%B5" title="Дом Фарнезе">Фарнезе</a>.</li>
<li>Май&#160;— <a href="/wiki/%D0%9F%D0%BB%D1%8E%D1%81%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D0%B5%D1%80%D0%B5%D0%BC%D0%B8%D1%80%D0%B8%D0%B5" title="Плюсское перемирие">Плюсское перемирие</a> между <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%BE%D0%B5_%D1%86%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Русское царство">Россией</a> и <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швецией</a>. Окончание <a href="/wiki/%D0%9B%D0%B8%D0%B2%D0%BE%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Ливонская война">Ливонской войны</a>. Присоединение к Швеции Ивангорода, Копорья, Яма, Корелы.</li>
<li>Гавань <a href="/wiki/%D0%A1%D0%B5%D0%BD%D1%82-%D0%94%D0%B6%D0%BE%D0%BD%D1%81_(%D0%9D%D1%8C%D1%8E%D1%84%D0%B0%D1%83%D0%BD%D0%B4%D0%BB%D0%B5%D0%BD%D0%B4_%D0%B8_%D0%9B%D0%B0%D0%B1%D1%80%D0%B0%D0%B4%D0%BE%D1%80)" title="Сент-Джонс (Ньюфаундленд и Лабрадор)">Сент-Джонс</a> и прилегающие территории на <a href="/wiki/%D0%9D%D1%8C%D1%8E%D1%84%D0%B0%D1%83%D0%BD%D0%B4%D0%BB%D0%B5%D0%BD%D0%B4" title="Ньюфаундленд">Ньюфаундленде</a> объявлены владениями <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D1%8F" title="Англия">Англии</a>.</li>
<li>Около 1583&#160;— <a href="/wiki/%D0%90%D0%B1%D0%B4%D1%83%D0%BB%D0%BB%D0%B0-%D1%85%D0%B0%D0%BD_II" title="Абдулла-хан II">Абдулла II</a> просил <a href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD_IV" class="mw-redirect" title="Иван IV">Ивана IV</a> о свободном пропуске его купцов в <a href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%B0%D1%85%D0%B0%D0%BD%D1%8C" title="Астрахань">Астрахань</a>. В ответ Иван IV разрешил торговлю и в других русских городах.</li>
<li>1583—1598&#160;— хан Бухары и всех <a href="/wiki/%D0%A3%D0%B7%D0%B1%D0%B5%D0%BA%D0%B8" title="Узбеки">узбеков</a><sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup> <a href="/wiki/%D0%90%D0%B1%D0%B4%D1%83%D0%BB%D0%BB%D0%B0-%D1%85%D0%B0%D0%BD_II" title="Абдулла-хан II">Абдулла II</a> (1534—1598), сын <a href="/wiki/%D0%98%D1%81%D0%BA%D0%B0%D0%BD%D0%B4%D0%B5%D1%80-%D1%85%D0%B0%D0%BD" title="Искандер-хан">Искандер-хана</a>.</li>
<li>Основаны города <a href="/wiki/%D0%9A%D0%BE%D0%B7%D1%8C%D0%BC%D0%BE%D0%B4%D0%B5%D0%BC%D1%8C%D1%8F%D0%BD%D1%81%D0%BA" title="Козьмодемьянск">Козьмодемьянск</a> и <a href="/wiki/%D0%90%D1%80%D1%85%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D1%8C%D1%81%D0%BA" title="Архангельск">Архангельск</a>.</li></ul>
